# BTree Business
Implementação em árvore B de um sistema de controle de fluxo financeiro para um empreendedor que deseja fazer o controle do fluxo financeiro de entradas e saídas. 
# Features
- As entradas e saídas podem ser à vista ou à prazo e serem feita com dinheiro, cheque, boleto ou cartão. 
- O sistema possui funções básicas de controle de dados e que ajudem a resolvê-lo. 
- O usuário odecide as operações que deseja fazer e o algoritmo tem que resolver essas operações escolhidas, exemplos de operações são pagamento, recebimento e consulta. - Ainda tendo que considerar que a loja estará funcionando por anos a fim, os registros que tiverem prazo maior que 12 meses tem de permanecer guardados em uma
outra árvore.
- Interface gráfica simples para utilização do sistema
